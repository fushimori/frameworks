# Машинное обучение: Прогнозирование успеха фильмов

## Описание проекта

Проект содержит 5 лабораторных работ по машинному обучению на данных о фильмах. Решаются две задачи:
- **Классификация**: Прогнозирование коммерческого успеха фильма
- **Регрессия**: Прогнозирование зрительского рейтинга

## Датасет

- Источник: The Movies Dataset (Kaggle)
- Размер: 5381 фильм
- Признаки: бюджет, популярность, длительность, жанр и др.
- Цели: успех (доход > бюджет × 1.5), рейтинг (0-9.1)

## Лабораторные работы

1. **KNN** - K-ближайших соседей
2. **Линейная и логистическая регрессия** - с регуляризацией
3. **Решающее дерево** - с визуализацией
4. **Случайный лес** - ансамблевый метод
5. **Градиентный бустинг** - последовательное улучшение + итоговое сравнение всех методов

## Метрики

- Классификация: Accuracy, Precision, Recall, F1, ROC-AUC
- Регрессия: MAE, RMSE, R²

## Запуск

Требования: Python 3.8+, pandas, numpy, scikit-learn, matplotlib, jupyter

Установка: `pip install pandas numpy scikit-learn matplotlib jupyter`

Запуск: `jupyter notebook` → открыть .ipynb файлы

## Результаты

Каждая работа включает базовую модель, оптимизацию, собственную реализацию и анализ.
